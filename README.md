# Portfolio Management Analysis (Excel)

## Project Overview
This project focuses on analyzing investment portfolio performance using Microsoft Excel.  
The goal of the report is to evaluate portfolio returns, asset allocation, and overall investment performance.

The dashboard provides a structured view of portfolio data and helps understand how different assets contribute to total portfolio performance.

---

## Objectives
- Analyze investment portfolio performance.
- Understand asset allocation and portfolio distribution.
- Track portfolio returns and performance trends.
- Create a simple and interactive financial dashboard.

---

## Dataset Description
The dataset contains information related to investment portfolios such as:

- Asset Name
- Asset Type
- Investment Value
- Portfolio Allocation
- Returns
- Performance Indicators

This data helps evaluate how each asset contributes to the overall portfolio.

---

## Excel Features Used
The project uses several Excel features for analysis and automation:

- Pivot Tables for summarizing portfolio data
- Charts for visualizing portfolio distribution
- Excel formulas for financial calculations
- Conditional formatting for highlighting performance trends
- VBA and Macros for report navigation and automation

---

## Dashboard Features
The Excel dashboard provides insights into:

- Portfolio distribution across different assets
- Investment performance comparison
- Portfolio allocation analysis
- Key financial metrics and summaries

The report allows users to quickly understand the performance of their investment portfolio.

---

## Automation with VBA & Macros
VBA and Macros were used to improve report usability by:

- Creating buttons for navigating between report sections
- Automating repetitive tasks
- Improving the user interaction with the dashboard

---

## Key Insights
The report helps identify:

- Which assets contribute most to portfolio returns
- How investments are distributed across asset types
- Portfolio performance trends over time

These insights support better investment decision-making.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- VBA
- Macros

---

## Project Outcome
This project demonstrates how Excel can be used to build a simple portfolio analysis dashboard to evaluate investment performance and support financial decision-making.
